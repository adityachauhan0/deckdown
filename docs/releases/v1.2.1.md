# DeckDown v1.2.1

DeckDown `v1.2.1` ships the first production-ready Studio editing upgrade on top of `v1.2.0`, with a stronger authoring surface, better project setup, and expanded diagram/math support.

## Highlights

- Studio editor upgraded to CodeMirror with:
  - syntax highlighting
  - DeckDown-aware snippets and autocomplete
  - inline diagnostics while typing
  - selection-driven slide sync
- Studio workspace and navigation upgraded with:
  - collapsible file tree
  - active-slide outline
  - top-bar docs view
  - smoother pane resizing
  - fixed-dimension preview stage for more reliable visual review
- New project creation flow for empty directories with templates for:
  - presentation `16:9`
  - presentation `4:3`
  - document `A4`
  - document `Letter`
  - custom page setup
- Added AI workflow surfaces:
  - repo-root `AGENTS.md`
  - `docs/ai.md`
  - `deckdown ai-prompt`
- Added LaTeX and Mermaid support across the Studio render pipeline.
- Fixed Mermaid `classDiagram` rendering so labels and members render correctly in the localhost preview and export path.

## Verification

Release verification for `v1.2.1` covered:

- full Jest suite
