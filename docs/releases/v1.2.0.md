# DeckDown v1.2.0

DeckDown `v1.2.0` turns the package from a compiler-only workflow into a usable local authoring loop for AI-agent and human-in-the-loop presentation work.

## Highlights

- `deckdown init` creates a repo-first workspace with a starter deck, theme, notes folder, assets folder, and local workspace metadata.
- `deckdown studio` now supports empty directories and can bootstrap a new workspace directly from the localhost UI.
- Studio now includes:
  - source-first Markdown editing
  - syntax-hint toolbar actions
  - live preview with stale-preview retention
  - actionable diagnostics
  - slide labels and active-slide context
  - local export actions for PDF, PNG, and PPTX
- Compiler validation now reports:
  - empty deck errors
  - missing title warnings
  - missing local image warnings

## Verification

Release verification for `v1.2.0` covered:

- full Jest suite
- sample PDF/PNG/PPTX rendering
- PDF and PPTX conversion parity checks
- npm packaging assertions for Studio and workspace files
- live localhost browser verification of:
  - existing-deck Studio flow
  - empty-directory bootstrap flow
