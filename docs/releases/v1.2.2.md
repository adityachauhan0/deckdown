# DeckDown v1.2.2

DeckDown `v1.2.2` builds on the Studio work from `v1.2.1` and focuses on polish: better source-first editing, better document preview, and a stronger release surface for the package itself.

## Highlights

- Added Studio quality-of-life upgrades:
  - inline asset preview
  - persisted layout and view preferences
  - keyboard shortcuts for save, preview, file drawer, slide navigation, and tab navigation
  - per-tab dirty/session tracking
- Added HTML-aware document preview for repo docs and `README.md` content:
  - safe rendering of README-style inline HTML blocks
  - local asset resolution through Studio
  - Studio-aware navigation from workspace document previews into local docs
- Fixed several Studio workflow issues around:
  - file handling and non-deck previews
  - export target stability
  - slide segmentation fidelity
  - bootstrap state refresh
- Refreshed the package presentation for `v1.2.2` with:
  - a fully rewritten README
  - a new two-slide showcase deck rendered by DeckDown itself
  - fresh localhost Studio screenshots

## Verification

Release verification for `v1.2.2` covered:

- full Jest suite
- full `npm run release-check`
- sample PDF/PNG/PPTX rendering
- npm packaging assertions
- live localhost Studio validation of:
  - the new release showcase deck
  - README/document preview rendering
  - Studio screenshot capture paths
